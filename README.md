# ![](https://user-images.githubusercontent.com/4441470/224455560-91ed3ee7-f510-4041-a8d2-3fc093025112.png) Soenneker.ServiceBus.Receptors.MsTeams

## Installation

```
dotnet add package Soenneker.ServiceBus.Receptors.MsTeams
```
